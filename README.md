## 🎶프로젝트 소개

프로젝트 명인 추노(CHUNO)에는 '**추**천하는 **노**래'라는 의미를 담아, 사용자들로 하여금 각자의 플레이스트를 공유하는 경험을 유도하고 있습니다. 사용자들은 자신의 플레이리스트를 추천할 수 있고, 동시에 타인의 플레이리스트를 감상할 수도 있습니다.

## 주요 기능

⭐ 유튜브 API를 사용하여 곡 서치, 등록, 재생 기능을 구현하였습니다. <br>
⭐ 원하는 카테고리에 따라 노래 리스트를 추천 받을 수 있습니다. <br>
⭐ 회원은 게시판을 통해 자신이 만든 노래 리스트를 추천 할 수 있습니다.<br>

<br>

## 구현 설명

✅ 로그인 / 회원가입 기능을 구현하였습니다. <br>
✅ 모듈 CSS를 사용해 페이지를 스타일링 하였습니다. <br>
✅ Recoil을 활용하여 상태관리 하였습니다. <br>
